(Recursive) ประสบการณ์ในสัปดาห์นี้มีการเชื่อมต่อกับประสบการณ์หรือแผนการที่ผ่านมาหรือไม่? ( ไม่มีการเชื่อมต่อกับประสบการณ์ที่ผ่านมาเนื่องจากได้เรียนเป็นครั้งแรก )
(CE) อะไรคือประสบการณ์ที่ผ่านมาของคุณ ? ( การใช้ Program Atom element html และภาษาเบื้องต้นที่ใช้ในการเขียนเว็บไซต์ )
(RO) อะไรคือประสบการณ์ที่สำเร็จและล้มเหลว ? (สร้าง Website อย่างง่ายเป็นของตัวเองได้สำเร็จ และล้มเหลวในการเขียนด้วย Dreamweaver เนื่องจากโปรแกรม Expert เกินไป )
(AC) คุณได้เรียนรู้อะไรจากประสบการณ์ ? ( ได้เพิ่มความรู้ในการเขียนเว็บไซต์ )
(AE) คุณจะได้นำสิ่งที่เรียนรู้จากประสบการณ์ ไปใช้อย่างไรต่อไป ? ( นำไปเป็นความรู้ในการศึกษาและฝึกงานในตำแหน่ง Front-End ในอนาคต )
